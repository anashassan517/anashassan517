### ü§ñ AI/ML Projects

#### [Vision LLM OCR Engine](https://github.com/anashassan517)
- **Tech:** LLMs, OCR, Computer Vision, PyTorch, CLIP, DocTR
- **Features:** Document understanding system that extracts structured information from scanned bank statements across all Pakistan banks
- **Impact:** Eliminated 80% manual effort in document processing, improved extraction accuracy to 92%, and reduced processing time from hours to minutes

#### [HR & Retail Chatbots](https://github.com/anashassan517)
- **Tech:** LangChain, FAISS, RAG, LLM, FAISS
- **Features:** Dual-purpose system with HR module for employee policy queries and retail module for customer support
- **Impact:** Achieved 90% first-contact resolution rate, reduced query handling time by 75%, and seamlessly integrated with bank's internal knowledge base

#### [ProtecTa - Multi-Factor Authentication System](https://github.com/anashassan517/)
- **Tech:** Python, React Native, Flask, Azure, OpenCV, InsightFace
- **Features:** Multi-factor authentication combining email, signature matching, facial recognition, and fingerprint validation
- **Impact:** Awarded Best Final Year Project Idea at FAST-NUCES

#### [Real-time ATM Anomaly Detection](https://github.com/anashassan517)
- **Tech:** YOLOv8, TensorRT, Triton Inference Server, GStreamer, Django, InsightFace
- **Features:** Detects suspicious behavior and masked individuals and recognize culprits at ATMs in real-time
- **Impact:** Deployed across 150+ ATMs with 65% latency reduction (600ms to <100ms)

#### [Recommendation Engine](https://github.com/anashassan517)
- **Tech:** PyTorch, Collaborative Filtering, Content-Based Filtering
- **Features:** Personalized recommendations based on user behavior and item similarity
- **Impact:** 22% increase in conversion rate for e-commerce platform

#### [RAG-Powered Compliance Chatbot](https://github.com/anashassan517)
- **Tech:** LangChain, FAISS, LLM, OCR
- **Features:** Question answering over banking compliance documents with 90% accuracy
- **Impact:** Reduced compliance query resolution time by 75%
